静态博客在线攥写文章不是梦。

*
{:toc}

## 功能

1. 自动生成yml头
2. markdown自动保存
3. 头图可直接预览
4. 随意添加或选择任意分类
5. 随意添加Tags
6. 更好的日期选择器
7. 随时编辑包括手机

## 使用

1. 打开 `https://yourblogurl/pages/editor.html`。
2. 按照提示填写对应信息。
4. 点击 `CREATE FILE` 按钮。
5. 点击 `Download` 按钮。
6. 将下载的文件移动到 `_posts` 文件夹。
7. `git push` 。
